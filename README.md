# 💧 Water Potability Prediction

This is my first Deep Learning project.

The application predicts whether water is safe for drinking using a Neural Network built with TensorFlow/Keras.

## Technologies
- Python
- TensorFlow
- Keras
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Features
- Predicts water potability
- Interactive Streamlit interface
- Real-time predictions
